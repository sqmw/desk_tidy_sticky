# Workspace 今日截止任务：超时优先与弱提醒

日期：2026-02-10  
范围：工作台左侧「今日截止任务」展示策略

## 变更目标
- 不增加打断式提醒（无弹窗、无强提示）；
- 让真正需要处理的任务先被看到；
- 用轻量视觉层级提醒超时状态。

## 具体改动
### 1) 排序策略改为“状态优先 + 紧迫度优先”
- 排序优先级：
  1. `已超时`
  2. `进行中`
  3. `待开始`
- 同状态内排序：
  - `已超时`：超时越久越靠前（`minutesLeft` 更小）
  - `进行中`：剩余时间越少越靠前
  - `待开始`：距离开始时间越近越靠前

### 2) 超时任务弱提醒视觉
- 保留原卡片样式基础上增加轻量层级：
  - 边框/背景红色弱混合
  - 状态徽标加红色小脉冲点
- 目标：可感知但不过度打扰。
