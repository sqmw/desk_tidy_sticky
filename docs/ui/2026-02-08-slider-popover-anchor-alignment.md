# 2026-02-08 透明度/磨砂滑条锚点对齐优化

## 问题
滑条弹层使用固定 `right` 定位，视觉上会偏离触发按钮，不符合“按钮在滑条中间”的预期。

## 修复
- 文件：`src/routes/note/[id]/+page.svelte`
- 方案：
  1. 为透明度与磨砂按钮增加 `tool-popover-anchor` 容器（`position: relative`）。  
  2. 将对应弹层放入该容器内部。  
  3. 弹层定位改为 `left: 50% + translateX(-50%)`，实现中心对齐按钮。

## 结果
透明度和磨砂滑条会以各自按钮为锚点居中弹出，位置更直观、操作一致性更好。
