# 2026-02-08 Sticky 底部工具栏悬浮遮罩交互

## 目标
默认显示更多正文内容；鼠标悬浮时再显示底部工具栏，并用遮罩降低底部文字干扰。

## 调整
- 文件：`src/routes/note/[id]/+page.svelte`
- 变更：
  1. 工具栏改为绝对定位覆盖层，不再占用正文布局高度。  
  2. 新增 `toolbar-mask`（底部渐变遮罩），仅在悬浮时显示，并提高遮挡强度。  
  3. 工具栏增加半透明底板、描边、阴影与轻微模糊，避免底层文字干扰图标可读性。  
  4. 悬浮时工具栏淡入并启用点击；非悬浮状态下工具栏隐藏且不接收指针事件。

## 结果
1. 非悬浮状态下底部区域可继续显示文本内容。  
2. 悬浮后底部文字会被明显遮挡，工具栏按钮不再受正文视觉干扰。
