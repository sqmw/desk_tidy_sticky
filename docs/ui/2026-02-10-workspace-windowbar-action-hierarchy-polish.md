# Workspace WindowBar 操作层级优化

## 目标
- 提升工作台右上角按钮的视觉分组与操作优先级。
- 保持原有功能不变，仅优化结构和交互可读性。

## 调整内容

### 1. 按钮分组改造
- 文件：`src/lib/components/workspace/WorkspaceWindowBar.svelte`
- 将右侧操作改为三个清晰分组：
  - 快捷组：主题切换
  - 模式组：返回简洁模式
  - 窗口组：最大化/还原 + 关闭（隐藏）

### 2. 操作语义增强
- “返回简洁模式”按钮增加返回箭头图标，减少文字孤立感。
- 各分组采用统一容器样式，层次更明确，符合常见桌面窗口习惯。

### 3. 一致性与响应式
- 统一按钮高度/圆角/间距：
  - 主按钮高度：`36px`
  - 图标按钮宽度：`36px`
- 右侧按钮区域禁用换行，避免高分辨率或窄宽下错层。
- 小屏下仅缩小间距与按钮内边距，不改变功能顺序。

## 验证
- 执行：`npm run check`
- 结果：`0 errors, 0 warnings`
