# 四象限视图体验优化（Pass 1）

## 目标
- 提升四象限视图的信息层级与可扫描性。
- 在不改数据结构的前提下，降低卡片噪音、突出优先级信息。

## 本次改动

### 1. 象限头部增加数量提示
- 文件：`src/lib/components/panel/WorkbenchSection.svelte`
- 每个象限标题右侧增加数量徽标，用户可快速判断负载分布。

### 2. 卡片密度与可读性优化
- 四象限卡片启用紧凑布局：
  - 降低卡片内边距与间距
  - 文本区域高度优化，避免长内容挤压操作区
  - 渐隐遮罩比例调整，减少文本截断突兀感

### 3. 操作区降噪
- 四象限卡片底部操作按钮默认弱显（低透明度）。
- 鼠标悬浮或键盘聚焦时完整显现。
- 结果：默认阅读态更干净，操作态依然可达。

## 受影响文件
- `src/lib/components/panel/WorkbenchSection.svelte`

## 验证
- 执行：`npm run check`
- 结果：`0 errors, 0 warnings`
