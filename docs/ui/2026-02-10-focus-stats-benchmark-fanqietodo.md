# 番茄统计能力对标（番茄ToDo）与落地（2026-02-10）

## 调研结论（公开信息）
基于番茄ToDo公开介绍与版本更新信息，可提炼的统计方向：
1. 专注时长与番茄数（今日/周期累计）。
2. 长周期趋势与热力图类展示。
3. 连续专注、里程碑/徽章等激励维度。
4. 年度/周期报告类汇总。

## 本项目本轮落地
在不破坏现有架构的前提下，先落地可直接由本地 `focusStats` 计算出的统计：
1. 连续专注天数（streak）。
2. 7天日均专注分钟。
3. 最近30天最佳单日（日期 + 分钟）。
4. 最近28天专注热力图。
5. 今日任务分布 Top（按任务番茄数排序）。
6. 原有统计保留：今日专注、今日番茄、最近7天专注/番茄、目标达成率。

## 架构调整
1. 新增纯逻辑分析模块：`src/lib/workspace/focus/focus-analytics.js`
   - 负责 streak / heatmap / best-day / average 计算。
2. `WorkspaceFocusHub` 只做编排与派发，不塞分析细节。
3. `WorkspaceFocusStats` 只负责展示，接收已计算结果。
