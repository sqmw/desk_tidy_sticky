# 四象限拖拽：50% 覆盖后直接占位规则（2026-02-10）

## 问题
- 用户反馈占位仍有滞后：拖到目标卡片上方时，空位没有及时挤出。

## 调整
- 将 50% 覆盖规则改为“直接占位”：
  - 当拖拽卡与某目标卡重叠比例 `>= 0.5` 时，
  - 占位索引直接定位到该目标卡位置（before），不再做 before/after 二次判定。

## 效果
- 占位出现更早、更稳定，符合“拖到某区域就预留空位，松手即落位”的直觉。

## 影响文件
- `src/lib/components/panel/WorkbenchSection.svelte`

## 验证
- `npm run check` 通过（0 error / 0 warning）。
